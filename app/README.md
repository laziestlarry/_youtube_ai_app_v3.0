Lean app base (symlinked)

This folder provides a distilled structure for production work without moving core directories.
All entries are symlinks to the current repo roots.

Structure:
- app/backend -> backend
- app/frontend -> frontend (Vite)
- app/frontend_v3 -> frontend_v3 (Autonomax dashboard)
- app/services -> services (API entrypoints)
- app/static -> static
- app/modules -> modules
- app/scripts -> scripts
- app/docs -> docs
- app/config -> config
