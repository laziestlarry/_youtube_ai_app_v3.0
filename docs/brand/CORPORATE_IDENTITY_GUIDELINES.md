## Voice & Tone

### Brand Voice Attributes

| Attribute | Description | Example |
|-----------|-------------|---------|
| **Conversational** | Like talking to a smart friend | "Here's what actually works..." |
| **Clever** | Smart without being smug | "The lazy way is the smart way" |
| **Practical** | Focused on results | "This saved me 10 hours/week" |
| **Encouraging** | Supportive, not pushy | "You can do this too" |
| **Honest** | Transparent about limitations | "This won't work for everyone" |

### Tone by Context

| Context | Tone | Example |
|---------|------|---------|
| Marketing | Energetic, benefit-focused | "Work less. Earn more." |
| Education | Patient, step-by-step | "Let's break this down..." |
| Support | Empathetic, helpful | "I understand the frustration..." |
| Technical | Clear, precise | "The API endpoint accepts..." |
| Social | Casual, engaging | "Anyone else automate their morning routine?" |

### Writing Guidelines

**DO**:
- Use active voice
- Keep sentences short
- Lead with benefits
- Use "you" and "we"
- Include specific numbers
- Tell stories

**DON'T**:
- Use jargon without explanation
- Make empty promises
- Sound corporate or stiff
- Use passive voice
- Be condescending
- Oversell

## Application Guidelines

### Website Design

**Layout**: Clean, spacious, focused on content

**Navigation**: Simple, maximum 5 main items

**CTAs**: Clear, benefit-focused, one primary per page

**Images**: Authentic, not stock-looking, show results

### Social Media

**Profile Images**: Consistent across all platforms

**Cover Images**: Brand-specific, updated quarterly

**Post Design**: 
- Use brand colors
- Consistent fonts
- Template-based for efficiency

### Email Communications

**From Name**: Lazy Larry or specific brand

**Subject Lines**: Benefit-driven, curiosity-inducing

**Format**: 
- Short paragraphs
- Bullet points for scannability
- Single CTA per email
- Personal sign-off

### Print Materials

**Business Cards**: 
- Minimal design
- QR code to link tree
- Brand colors

**Documents**:
- Clean headers
- Consistent margins
- Brand footer

Partners must receive written approval and follow co-branding guidelines.

### Public Use
Media and press may use logos with proper attribution. Modified versions require approval.

### Prohibited Use

- Using brand assets for unrelated products
- Implying endorsement without agreement
