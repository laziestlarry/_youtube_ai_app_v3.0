# Customer Journeys

These journeys are designed to be clear, engaging, and conversion-focused.

## 1) Shopier Direct (Consulting + Training)
1. Landing page -> pain point narrative
2. Offer overview -> deliverables
3. CTA -> Shopier payment
4. Confirmation -> onboarding email + intake form
5. Delivery -> milestone updates -> final handoff

## 2) Shopify Digital Products
1. Product landing -> benefits + preview
2. Bundle upsell -> add to cart
3. Checkout -> instant delivery
4. Follow-up -> review request + cross-sell

## 3) Etsy Digital Products
1. Search discovery -> strong cover image
2. Listing -> SEO tags + quick value summary
3. Purchase -> instant download
4. Review request + bonus asset

## 4) Fiverr Services
1. Gig discovery -> portfolio proof
2. Packages -> compare scope
3. Order -> requirements intake
4. Delivery -> milestone updates
5. Review request + retainer offer

## 5) YouTube Lead Engine
1. Content -> short value proof
2. CTA -> lead magnet / store link
3. Email nurture -> product + service
4. Conversion -> Shopier / Shopify / Fiverr
