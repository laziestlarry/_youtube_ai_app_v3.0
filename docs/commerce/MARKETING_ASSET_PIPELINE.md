# Marketing Asset Pipeline

This pipeline prepares listing visuals and metadata for Shopify, Etsy, Fiverr, Amazon, and Gumroad uploads.

## Inputs
- `docs/commerce/product_catalog.json`
- Listing assets generated by `scripts/generate_listing_assets.py`

## Outputs
- `marketing_assets/index.json` (full manifest)
- `marketing_assets/channel_shopify.csv`
- `marketing_assets/channel_etsy.csv`
- `marketing_assets/channel_fiverr.csv`
- `marketing_assets/channel_amazon.csv`
- `marketing_assets/channel_gumroad.csv`
- `marketing_assets/channel_shopier.csv`

## Run
```bash
python3 scripts/generate_listing_assets.py \
  --catalog docs/commerce/product_catalog.json

python3 scripts/prepare_marketing_assets.py \
  --catalog docs/commerce/product_catalog.json \
  --output marketing_assets \
  --backend-url https://your-backend-domain
```

## Notes
- The pipeline does not upload to marketplaces directly.
- Use the generated CSV files as upload templates.
- Add channel-specific assets to `marketing_assets/<SKU>/` if needed.
- When `--backend-url` is provided, local `static/` image paths are converted to absolute URLs.
