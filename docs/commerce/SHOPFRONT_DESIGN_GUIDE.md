# Shopfront Design Guide (Etsy + Shopify)

This guide defines the storefront design, copy, and asset usage for Etsy and Shopify.
Assets referenced are already in the repo for fast upload.

## Visual System
- Primary style: minimalist, calm, premium.
- Typography: clean sans serif (use default Etsy/Shopify).
- Palette: warm neutrals + charcoal + gold accents.

### FAQ (Etsy)
- What do I receive? High-resolution JPG/PNG and a printable PDF bundle.
- Is this a physical item? No, all items are digital downloads.
- Can I print at home? Yes, home, local print shop, or online print services.
- Refunds? Digital downloads are non-refundable once delivered.
- Commercial use? Personal use only unless stated otherwise.

### Policies (Etsy)
- Digital delivery: instant upon purchase.
- Returns/exchanges: not accepted for digital items.
- Support: message response within 24 hours.
