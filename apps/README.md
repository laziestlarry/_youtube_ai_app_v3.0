# Apps

Product-level entry points. Each app folder contains symlinks to its canonical code so the repo stays single-source.

## Autonomax
- `apps/autonomax/api` -> `services/autonomax_api`
- `apps/autonomax/ui` -> `frontend_v3`
- `apps/autonomax/ops` -> `autonomax`
- `apps/autonomax/static` -> `static`

## YouTube AI
- `apps/youtube-ai/api` -> `services/youtube_ai_api`
- `apps/youtube-ai/ui` -> `frontend`
- `apps/youtube-ai/static` -> `static`
